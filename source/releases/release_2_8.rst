Release Notes
=============

.. attention::

   The 2.8 Beta 1 release is the next release and in development.

   You can follow the development progress on the `2.8 Beta 1 Milestone <https://github.com/vkbo/novelWriter/milestone/89>`__.


Planned Features
================

Here are some planned features for 2.8. The plan is subject to change.

* Revamped theme system: The GUI and document themes will be merged.
* Quick dark/light theme switching.
* Writing session progress counter and progress bar.
* EPUB export format.
* Character dialogue statistics for the manuscript.
